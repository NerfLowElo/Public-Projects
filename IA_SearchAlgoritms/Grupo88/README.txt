Para gerar um conjunto de pontos é preciso compilar e executar a classe Generator.java. Esta classe precisa (como input) de dois inteiros que representam o nº de pontos e o limite das suas coordenadas respectivamente. A classe assegura que nao existem pontos iguais
Para executar os algoritmos é preciso compilar e executar  a classe MainIa.java. O output gerado pela classe generator serve como intput para esta, se quiser introduzir input manual a classe recebe um inteiro n que sao o nº de pontos e n*2 inteiros que sao as coordenadas dos pontos
Para vizualizar graficamente os grafos é só abrir o ficheiro graph.html no browser, inserir um ficheiro.txt com o output da classe MainIa onde diz browse e inserir o limite dos valores dos pontos onde diz choose graph limit e depois escolher qual o grafo que quer ver. Pode alternar entre grafos sem ter de dar refresh á pagina
